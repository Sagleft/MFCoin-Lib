
# MFCoin-Lib
C# библиотека для работы с кошельком, транзакциями и адресами.
Осуществляет подключение к MFCoin кошельку через JSON-RPC.
Методы документированы.

## Зависимости
Зависит от:
* [Newtonsoft.Json.dll](https://www.newtonsoft.com/)
